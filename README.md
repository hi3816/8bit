# :pouting_cat: 8bit 
이번 프로젝트는 2D 플랫포머 게임으로 쇼본의 액션(***일명 고양이 마리오***) 로 유명한 게임을 레퍼런스로 다양한 함정이 도사리는 게임을 제작하려 합니다.

# :couple: 프로젝트 소개
유니티를 이용해 5인이 협력하여 게임을 개발하는 프로젝트 입니다.
약 일주일간 진행되며 해당기간동안 기획된 기능을구현하는것이 최우선 목표입니다.
